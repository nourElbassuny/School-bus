# School-Bus
Parent Android app for School Bus Tracking System Project.

## Features
<ul type="disc">
  <li>Login</li>
  <li>List of Kids</li>
  <li>Upload Photo</li>
  <li>Add Home Gps</li>
  <li>Edit information of your student</li>
  <li>Remove your student from bus</li>
  <li>Driver can find your home easily</li>
</ul>
